Welcome to the A-Maze-Ing Game!

    Game Rules:

  • Use the w-a-s-d keys to move your character around the maze.
  • Allowed 3 peeks at the full maze
  • Time is limited based on the size of the maze. The larger the maze, the more difficult!
  • Pressing Reveal will display the full maze, and lose the game.

Please select a maze size

The A-Maze-Ing Game is a random maze generator turned online game! The generator uses Kruskal's algorithm to generate a random maze of the given width and height. Enter your dimensions and press create to start the game.
